# 🎨 Demonstration & Sandbox Notebooks

This directory contains standalone notebooks and assets designed to illustrate how to fetch and manipulate satellite and climate datasets. Use these notebooks to understand the underlying APIs before running the main pipeline.

## 📁 Directory Contents

* **`landsat_demo.ipynb`**: A lightweight sandbox notebook to demonstrate querying the Microsoft Planetary Computer STAC Catalog, signing URLs, and visualizing Landsat-8 surface reflectance bands.
* **`terraclimate_demo.ipynb`**: Demonstrates how to open, slice, and plot monthly climate parameter netCDF files from the TerraClimate datasets.
* **`terraclimate_parameters.png`**: Reference visualization illustrating the diverse weather and soil moisture variables provided by TerraClimate.
